36輪海選賽後，根據戰積晉級積分賽

  1. 比賽日，玩家需進入比賽服中進行匹配。
  2. 玩家開始匹配後，系統將自動分配對手進行5v5對戰。
  3. 每輪比賽都是一局定勝負，所有戰隊皆需經過36輪比賽。
  4. 非比賽期間，無戰隊玩家可自由加入戰隊參賽。

海選賽晉級128支戰隊，分為16個小組
進行組內單循環比賽，
最終32支戰隊進入淘汰賽

  1. 比賽開始前，系統會生成對陣名單。
  2. 比賽日，玩家需進入比賽服等待開賽。
  3. 組內單循環比賽，採三局兩勝制。
  4. 小組賽後，前兩名戰隊晉級。
  5. 未在入場時間內進入準備場景的玩家， 將自動視為棄權，對手直接獲勝。
  6. 排名積分相同時，按淨勝局、陣亡數、 平均戰力排名。


32支戰隊進行5輪淘汰賽

  1. 比賽開始前，系統會生成對陣名單。
  2. 比賽日，玩家需進入比賽服等待開賽。
  3. 所有戰隊皆需進行5輪淘汰賽， 前3輪為3戰2勝。
  4. 半決賽及決賽為5戰3勝比賽， 依次決出4強、亞軍、冠軍。
  5. 未在入場時間內進入準備場景的玩家， 將自動視為棄權，對手直接獲勝。
  6. 排名積分相同時，按淨勝局、陣亡數、 平均戰力排名。
